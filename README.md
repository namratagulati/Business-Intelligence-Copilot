# AI Business Intelligence Copilot
An AI-powered predictive analytics dashboard built using Python, Streamlit, Machine Learning, and LLM-based business insight generation.

---

# Features

- CSV dataset upload and analysis
- Automated data preprocessing
- Machine learning prediction pipeline
- Random Forest Classification
- Feature importance analysis
- Correlation matrix visualization
- Business priority segmentation
- AI-generated business insights using LLMs
- Interactive analytics dashboard

---

# Tech Stack

- Python
- Streamlit
- Pandas
- Scikit-learn
- Plotly
---

# Machine Learning Workflow

1. Dataset Upload
2. Data Cleaning
3. Feature Engineering
4. Label Encoding
5. Train/Test Split
6. Random Forest Model Training
7. Prediction & Accuracy Evaluation
8. Feature Importance Analysis
9. AI Insight Generation
